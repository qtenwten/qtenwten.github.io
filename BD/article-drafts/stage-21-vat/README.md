# Stage 21 — VAT Calculator Draft Articles

These files are reviewable draft article JSONs for the VAT calculator cluster.

Before publishing, selected files are copied to `BD/content-staging/`.
Publishing only happens after user approval.

All articles in this folder have `status: "draft"` and must remain draft until the publish stage.

Files:
- 3 RU draft articles
- 3 EN draft articles
- This README

Workflow:
1. Review drafts in this folder
2. User approves for publishing
3. Copy approved files to `BD/content-staging/` with `status: "published"`
4. Run `publish-article.ps1` for each
5. Rebuild after all 6 published
