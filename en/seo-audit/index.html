Helpful Guide

What the quick SEO audit checks

This quick audit checks visible on-page elements: title, meta description, H1/H2, images without alt text, Open Graph, and JSON-LD. It does not replace a full SEO strategy, but it helps catch obvious technical issues fast.

What is checked

  • Title and meta description presence and approximate length
  • H1 and H2 structure on the page
  • Images without alt attributes
  • Open Graph tags for social previews
  • JSON-LD structured data when present

Limitations

The basic audit runs from the browser, so some external websites may block requests. If you need to check a public site without CORS limits, use SEO Audit Pro.

How to read the score

  • 80-100: the basic technical elements look solid; next, review text quality and search intent.
  • 60-79: the page works, but several signals should be improved.
  • 0-59: fix title, description, H1, alt text, and basic structure first.

FAQ

Why can some sites not be checked?

The basic audit runs from your browser, and many websites block cross-origin requests. SEO Audit Pro uses server-side checking for public URLs.

Does this replace a full SEO audit?

No. It is a fast technical checklist for common on-page issues before publishing or after editing a page.

Is the score a ranking factor?

No. The score is an internal indicator based on basic checks, not a metric from Google or Yandex.

Can I use it for drafts?

Yes, if the page is reachable from your browser. For private drafts, run it in the same browser session where you can open the page.
